Para iniciar el juego ejecutar el archivo PinguGame.exe
Tienes 3 vidas y cada vez que toque a un enemigo te quita un vida
las monedas son puntuacion aunque aun no se muestras
El boton escape te saca del juego
------------------------------------------------------
CONTROLES:
	W/flecha arriba = saltar
	A/flecha izquierda = mover a la izquierda
	D/flecha derecha = mover a la derecha
	Espacio = atacar
